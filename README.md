## 📊 일반 환경 vs 랜덤 벽 환경 상태가치(${V(s)}$) 비교 분석

### 1. 가치 절대값의 하락 (Magnitude of Values)
가장 큰 차이는 시작점인 $V(0, 0)$에서 나타나는 **음수 값의 깊이**입니다.

* **일반 환경:** 장애물이 없으면 무작위 정책이라도 목표 지점(${3, 3}$)에 도달하는 경로가 비교적 짧습니다. 보통 $V(0,0)$은 **-50 ~ -70** 사이에서 형성됩니다.
* **랜덤 벽 환경:** 현재 결과값인 **-102.66**은 일반 환경보다 훨씬 낮은 가치를 보여줍니다. 이는 벽에 가로막혀 제자리에서 `-1`의 보상을 반복해서 받는 '지연 시간'이 가치 함수에 그대로 누적되었기 때문입니다.

### 2. 이동 효율성과 패널티 기댓값
강화학습에서 상태가치는 해당 칸에서 기대할 수 있는 '**앞으로의 총 보상**'을 의미합니다.

* **일반 환경:** 이동 액션이 곧 위치 변화로 이어지므로, 가치는 순수하게 목표까지의 거리(Manhattan Distance)에 비례하여 깔끔하게 감소합니다.
* **랜덤 벽 환경:** 매 에피소드마다 2~3개의 벽이 랜덤하게 배치됩니다. 에이전트는 특정 칸에서 올바른 방향으로 움직이려 해도 **벽에 부딪혀 제자리에 머물 확률**이 생깁니다. 이 불확실성이 반영되어 모든 칸의 가치가 일반 환경보다 하향 평준화됩니다.

### 3. 가치 기울기(Value Gradient)의 불규칙성
목표 지점 근처의 수치를 보면 가치가 급격히 변하는 구간이 존재합니다.

* **데이터 관찰:** $(3, 2)$의 가치는 **-52.57**이고, $(2, 3)$의 가치는 **-45.78**입니다. 두 칸 모두 목표인 $(3, 3)$ 바로 옆이지만 가치 차이가 발생합니다.
* **원인:** 이는 랜덤하게 생성되는 벽의 위치 때문입니다. 특정 칸이 다른 칸보다 벽에 둘러싸일 확률이 높거나, 목표로 가는 길목이 차단될 확률이 높을 경우 해당 지점의 가치는 주변보다 더 가파르게 떨어집니다.

### 📝 요약 비교표

| 구분 | 일반 그리드 환경 (Clean) | 랜덤 벽 그리드 환경 (Wall) |
| :--- | :--- | :--- |
| **$V(0,0)$ 가치** | 높음 (약 -60) | **매우 낮음 (-102.66)** |
| **이동 성공률** | 100% (격자 끝 제외) | **낮음 (벽 충돌 확률 존재)** |
| **가치의 의미** | 목표까지의 순수 거리 기댓값 | **순수 거리 + 벽에 의한 지연 비용** |
| **업데이트 공식** | $V(s) \leftarrow V(s) + \alpha [R + \gamma V(s') - V(s)]$ | (좌동, 단 $V(s')$에 벽 충돌 케이스 포함) |
